## 📱 **视频制作建议**

### **1. 视频制作工具**
- **剪映**（手机/电脑，推荐）
- **CapCut**（手机）
- **Adobe Premiere**（专业）
- **DaVinci Resolve**（免费专业）

### **2. 录屏工具**
- **Windows**: Win + G (Xbox Game Bar)
- **macOS**: Cmd + Shift + 5
- **OBS Studio**（跨平台，专业）

### **3. 录音建议**
- 使用麦克风录制配音
- 或者用AI配音：
  - 剪映自带AI配音
  - Microsoft Azure TTS
  - OpenAI TTS-1

### **4. BGM推荐**
- **科技感**: 电子音乐、Synthwave
- **节奏**: 120-140 BPM
- **音量**: 背景音乐音量要低，不要盖过人声

### **5. 字幕样式**
- 字体：思源黑体、微软雅黑
- 颜色：白色文字 + 黑色描边
- 位置：底部或关键信息旁
- 动画：淡入淡出
