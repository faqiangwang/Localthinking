## 🎛️ **音量设置建议**

### **音量平衡**
```
背景音乐: 20-30%
配音/人声: 100%
音效: 40-60%
```

### **不同平台音量**
- **B站**：背景音乐可以稍大（25-30%）
- **抖音**：音乐适中（20-25%）
- **YouTube**：音乐稍小（15-20%）

## 🎬 **音效使用时机**

### **开场（0-5秒）**
- 0s: 鼠标点击音
- 1s: 界面切换音（×3）
- 4s: 问题出现音（低沉）

### **启动（5-15秒）**
- 5s: 文字显现音（嗖嗖）
- 8s: Logo出现音（清脆）
- 10s: 界面展开音（流畅）

### **模型管理（15-25秒）**
- 15s: 界面切换音
- 17s: 列表滚动音（轻微）
- 20s: 下载开始音
- 23s: 高亮音（提示）

### **对话演示（25-40秒）**
- 25s: 输入打字音
- 28s: 思考块展开音（重要）
- 32s: 折叠音
- 35s: 回答块出现音

### **功能展示（40-50秒）**
- 40s: 界面切换音
- 42s: 滑块移动音（×4）
- 46s: 按钮点击音（×3）
- 48s: 切换音（×2）

### **技术栈（50-55秒）**
- 50s: Logo出现音（×4）
- 53s: 代码滚动音

### **结尾（55-60秒）**
- 55s: 二维码出现音
- 58s: Star按钮音（重要）
- 60s: 结束音（有力）

## 🎛️ **音频处理建议**

### **音频编辑软件**
- **Audacity**（免费）
- **Adobe Audition**（专业）
- **GarageBand**（Mac免费）

### **音频处理技巧**
1. **降噪** - 去除背景噪音
2. **标准化** - 统一音量
3. **压缩** - 平衡动态范围
4. **均衡器** - 调整频率

### **导出设置**
- **格式**: MP3 或 AAC
- **比特率**: 128-192 kbps
- **采样率**: 44.1 kHz 或 48 kHz
- **声道**: 立体声

## 💡 **创意建议**

### **音乐节奏同步**
- 在关键切换点配合音乐节拍
- 使用"踩点"剪辑技巧
- 思考块展开时音乐上扬

### **音效增强**
- 思考块展开使用"魔法"音效
- 下载完成使用"叮"的提示音
- Star按钮使用"庆祝"音效

### **静音运用**
- 关键时刻（如思考块）短暂静音
- 突出重要信息
- 创造节奏感
