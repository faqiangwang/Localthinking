# 应用截图说明

## 需要添加的截图

请按照以下要求添加应用截图：

### 1. main-interface.png
**主界面截图**
- **尺寸**: 1200x800 像素
- **内容**: 完整的应用主界面
- **要求**:
  - 显示左侧边栏（会话列表）
  - 显示聊天区域
  - 包含一条示例对话
  - 界面清晰，无遮挡

### 2. chat-interface.png
**对话界面截图**
- **尺寸**: 1200x800 像素
- **内容**: 展示思考块功能的对话
- **要求**:
  - 包含完整的思考块（灰色背景）
  - 包含回答块
  - 展示折叠/展开状态
  - 最好有代码示例

### 3. model-manager.png
**模型管理界面截图**
- **尺寸**: 1200x800 像素
- **内容**: 模型推荐和管理界面
- **要求**:
  - 显示模型推荐列表
  - 显示已安装的模型
  - 展示不同Tier的推荐结果

### 4. settings.png
**设置页面截图**
- **尺寸**: 1200x800 像素
- **内容**: 设置界面
- **要求**:
  - 显示参数调节滑块
  - 显示系统提示词预设
  - 展示性能监控部分

## 截图工具推荐

### Windows
- **Snipping Tool** (内置)
- **Win + Shift + S** (截图快捷键)
- **ShareX** (功能强大)

### macOS
- **Command + Shift + 4** (区域截图)
- **Command + Shift + 5** (截图工具)

### Linux
- **Flameshot** (推荐)
- **Screenshot** (内置)
- **Shutter**

## 截图技巧

1. **使用浅色主题** - 更适合文档展示
2. **隐藏个人信息** - 不要显示敏感信息
3. **保持界面整洁** - 关闭不必要的窗口
4. **使用示例数据** - 展示有意义的内容
5. **统一尺寸** - 所有截图保持相同比例
6. **优化文件大小** - 使用PNG格式，适当压缩

## 添加截图

将截图文件放在此目录下，命名为：
- `main-interface.png`
- `chat-interface.png`
- `model-manager.png`
- `settings.png`

然后提交到Git：
```bash
git add docs/screenshots/
git commit -m "docs: 添加应用截图"
git push
```
